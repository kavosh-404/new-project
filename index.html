Introduction

What this website does

  • This website lets you run a simulation and watch matching happen in real time.
  • The simulation is a simple digital world where virtual people meet and decide yes or no.
  • Use it to learn how settings change pair results, search time, and match quality.
  • After each run, you get clear results you can cite and export for school or research.

Before You Start

  • Behind the scenes, the computer creates virtual people called agents.
  • An agent is one simulated person on the screen.
  • Each step, agents move, look around, and decide yes or no.
  • A match happens only when both sides say yes.
  • When you change settings, you change behavior, so results change too.

Settings Explanation

  • Preference Rule: changes how agents choose partners.
  • Mobility: higher means more movement and more meetings.
  • Density: higher means a more crowded space and faster encounters.
  • Selectivity: higher means pickier choices and more rejection.
  • Patience: controls how quickly standards relax during long search.
  • Exploration: wider search checks more places instead of staying local.
