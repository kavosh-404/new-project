Model Overview

Kalick & Hamilton (1986)

  • Well-mixed population; no spatial constraints.
  • Random encounters among all individuals.
  • Matching emerges from preferences + simple rules.
  • Everyone has equal access to all potential partners.
  • Assumes fixed attractiveness-based preferences.

Smaldino & Schank (2012)

  • Agents live on a 2D spatial grid.
  • Local neighborhoods restrict encounter opportunities.
  • Mobility changes partners over time.
  • Density affects encounter rates and search difficulty.
  • Satisficing: agents relax after repeated failures.
  • Ability to switch between attractiveness- and similarity-based mate choice.

Paper-Aligned Rule Analytics

Metric definitions
  • r Inter-pair correlation: Pearson correlation between attractiveness values of matched partners.
  • t_m Mean date to mate: Average step where matched agents formed pairs.
  • h(t) Mating hazard at step t: matches(t) / atRisk(t).

Statistics: each row reports mean with 95% CI from n=20 synthetic runs. CI colors: green=tight, amber=medium, rose=wide.

Code guide
  • R1 = Rule 1 (Attractiveness-based)
  • R2 = Rule 2 (Similarity-based)
  • NS = local neighborhood search, ZZ = balanced zig-zag search, BR = broad random search
